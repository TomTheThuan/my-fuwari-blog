## 其他项目

[as-biology-selfnotes](https://gitee.com/tomthethuan/as-biology-selfnotes)：我目前把我上课的AS生物课程的笔记放在这个地方，以后应该会慢慢完善。

[Generate Prompt with Milvus Database](https://gitee.com/tomthethuan/generate-prompt-with-milvus-database)：这是一个基于Milvus向量数据库的文档检索脚本，能够将Markdown文档分割、嵌入向量并存储，支持基于语义相似度的查询。

[Wenku8Downloader-TTT-modified](https://gitee.com/tomthethuan/wenku8-downloader-ttt-modified)：原Wenku8Downloader的修改版，根据我自己的需要添加了一些功能，包括自定义了EPUB的css样式、为书本添加元数据、批量下载、处理一些下载时遇到的错误等等。

[godot_1bit_platformer](https://gitee.com/tomthethuan/godot_1bit_platformer)：用godot游戏引擎做的一个平台跳跃游戏，目前算是把一些最基础的东西都写好了。目前只有四个用于展示功能的房间，但是可以很方便地添加新的内容和关卡。

[godot_2025-7-6-platformer](https://gitee.com/tomthethuan/godot_2025-7-6-platformer)：这是我另一个项目godot_1bit_platformer:的重写版，主要是通过状态机的方式实现了玩家的移动，完全没有设计关卡。可以把这个东西当作是一个平台跳跃游戏的模板。

[Gitee Release Automation Scripts](https://gitee.com/tomthethuan/gitee-release-automation-scripts)： Automated scripts for Git operations and Gitee release management
